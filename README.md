## SAFE TUNNEL

A secure end-to-end encryption chat application, that support channels.
It uses AES-256-CBC to encrypt the messages and RSA-4096 to encrypt the keys.


#### Starting the server.
```
node server.js
```


#### Running a client.
```
npm run dev
```
